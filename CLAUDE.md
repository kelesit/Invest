# 项目愿景

最终目标是建立一个能承载任意因子、任意品种、任意市场的量化研究和交易系统。CTA 趋势跟踪是入口，但不是终点——未来会扩展到截面动量、基本面信号、另类数据、多因子融合。

## 当前阶段

从 CTA 入手，边实践边学习。遇到问题时按需补充理论，不预先大量读书。

### 项目路线图

**项目一：回测可信度验证 — 已完成**
问题：我怎么知道回测结果不是碰巧好的？
- 数据分 2019-2023（训练）和 2024-2026（验证），看样本外是否还赚钱
- 参数扫描：lookback 从 5 到 120 全扫一遍，画热力图，看是普遍赚钱还是只有某个参数赚
- 随机基准：生成 1000 条随机信号跑回测，看策略排第几
- 产出：一套验证工具，之后每研究一个新信号都能用

**项目二：信号研究框架 — 下一步**
问题：除了动量还有什么趋势信号？它们有什么区别？
- 实现 5-6 种经典趋势信号（通道突破、MACD、回归斜率等）
- 建立统一评估：每个信号的夏普、相关性、不同市场环境表现
- 研究信号组合方式
- 产出：对趋势度量的完整认知，以及一个可扩展的信号库

**项目三：风控与组合优化 — 未开始**
问题：怎么把多个品种聪明地组合在一起？
- 动态相关性估计、风险平价权重、止损、波动率择时
- 产出：理解"风控比信号更重要"的原因，组合夏普显著提升

**项目四：跨市场扩展（A 股期货） — 未开始**
问题：同一个策略在不同市场表现一样吗？
- 接入 A 股期货数据，扩展到 10+ 品种
- 产出：双市场覆盖，理解市场微观结构差异

**项目五：CTA + 多因子融合 — 未开始**
问题：怎么把 CTA 和多因子结合成一个完整体系？
- 加入 carry（展期收益）、价值等横截面因子
- 产出：最初规划的 CTA + 多因子体系成型
