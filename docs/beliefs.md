## Lessons

### L1. 止损降回撤不升夏普

右尾和左尾被同时切掉。止损是风险偏好调节器，不是 alpha 来源。期待止损**同时**改善回撤和夏普是初学者的典型错觉。

_来源：项目三，4 CTA 品种。追踪止损让 Sharpe 从 0.91 → 0.67，MaxDD 从 -9.5% → -6%。_

### L2. ATR sizing 本身就是一层风控

ATR sizing 已经把每个品种归一化到单位波动率。再叠加波动率择时、组合波动率目标，是在同一个风险维度上做二次调整，没有增量信息，只会让行为变得更难归因。

**加风控前必须先问**：这一层是否在响应已经被响应过的信号？

_来源：项目三。_

### L3. 低相关不能救一个负期望因子

carry 信号与趋势信号相关 0.03–0.16（远低于趋势间的 0.63），看起来是理想的组合对象。但 carry 在 4 CTA 品种上单独 Sharpe -1.30，组合后直接拖累趋势。

**组合的第一前提：每个组件本身有正期望收益。相关性是第二顺位优化，不能逆转第一前提。**

_来源：项目四。代价明显——这是我见过自己犯的最典型的"漂亮数学覆盖坏前提"的错。_

### L4. 同一数据源的特征变体无法真正分散化

10 种趋势信号（动量 / 均线 / MACD / 回归斜率 / ADX / 通道突破 / ...）本质都是对价格时间序列的变换，两两相关 0.5–0.9。

**跨变换算子的组合 ≠ 跨数据源的组合。** 真正的分散化需要跨数据源（量价 / 财务 / 另类），而不是跨同一数据源的算子。

_来源：项目二。_
