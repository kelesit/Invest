# 趋势信号原理详解

## 核心思想

所有趋势信号都在回答同一个问题：**价格在涨还是在跌？趋势有多强？**

但它们用不同的方式来回答，各有优劣。可以分成四大类：

```
动量类 ── 直接看价格变化（涨了就做多）
均线类 ── 用平滑后的价格判断方向（去噪音）
通道类 ── 看价格是否突破了历史区间
回归类 ── 用数学拟合来度量趋势
```

---

## 一、动量类

### 1. Momentum（时间序列动量）

**一句话**：过去 N 天涨了就做多，跌了就做空。

**计算**：
```
信号 = sign(今天的价格 / N天前的价格 - 1)
```

**直觉**：这是最朴素的趋势信号——如果一个东西过去在涨，赌它继续涨。

**优点**：简单、直接、不需要调太多参数。
**缺点**：只看起点和终点，忽略中间的路径。如果过去 20 天先涨后跌但净变化为正，它会说"做多"，但趋势可能已经反转了。

**参数**：`lookback` — 回看天数。短（5-10天）= 快速反应但噪音多；长（60-120天）= 稳定但反应慢。

**本项目实现**：与经典定义一致，无修改。输出离散值 +1/-1/0。

---

### 2. Combined Momentum（多周期动量）

**一句话**：同时看短期、中期、长期动量，投票决定方向。

**计算**：
```
信号 = mean(momentum_12天, momentum_30天, momentum_60天)
```

**直觉**：单一周期的动量可能被噪音干扰。如果短中长期都看多，更可能是真趋势；如果短期看多但长期看空，说明趋势不一致，应该减仓。

**输出范围 [-1, +1]**：
- +1.0 = 三个周期都看多（强趋势共识）
- +0.33 = 两个看多一个看空（弱共识）
- 0 = 完全分歧

**本项目实现**：这个信号本身不是经典教科书里的标准定义，而是一种常见的工程实践——把多个周期的动量信号做等权平均。它是代码里唯一一个"原生连续值"的信号。

---

### 3. MACD（Moving Average Convergence Divergence）

**一句话**：快慢均线之间的距离在扩大还是缩小。

![MACD 原理图](assets/macd_explained.svg)

**先看图怎么读**：
- 上半部分先看价格、EMA12、EMA26。价格拐头向上时，快线先动，慢线后动，所以两条线的距离会先扩大。
- 中间公式链说明了 MACD 的三层结构：`快慢均线之差 -> 这个差的平滑值 -> 两者再做差`。
- 下半部分最关键：MACD 线告诉你方向，柱状图告诉你"方向变化是不是在加速"。

**经典定义**：
```
MACD线 = EMA(12天) - EMA(26天)        ← 快慢均线的差
信号线 = EMA(MACD线, 9天)              ← MACD线本身的均线
柱状图 = MACD线 - 信号线               ← 差值的差值
```

**把公式完全展开**：
```
EMA_t = α × Price_t + (1 - α) × EMA_{t-1}
其中 α = 2 / (N + 1)

所以：
EMA12_t = (2/13) × Price_t + (11/13) × EMA12_{t-1}
EMA26_t = (2/27) × Price_t + (25/27) × EMA26_{t-1}

MACD_t = EMA12_t - EMA26_t

Signal_t = (2/10) × MACD_t + (8/10) × Signal_{t-1}
         = 0.2 × MACD_t + 0.8 × Signal_{t-1}

Histogram_t = MACD_t - Signal_t
```

**这串公式怎么理解**：
- `EMA12` 给今天价格更高权重，所以反应更快。
- `EMA26` 给历史更高权重，所以更平滑、更慢。
- `MACD = EMA12 - EMA26`，表示短期趋势相对长期趋势领先了多少。
- `Signal` 是 `MACD` 自己的 9 日 EMA，相当于给动能再做一次平滑。
- `Histogram` 则是在看：当前动能相对"平滑后的平均动能"是更强还是更弱。

**初始化怎么做**：
- 实现里常见两种方法：第一天直接令 `EMA_1 = Price_1`，或者先用前 `N` 天的简单平均作为第一个 EMA。
- 只要样本足够长，前几天初始化的差异通常会很快衰减掉。

**经典用法**：
交易者通常看三种信号：
1. 柱状图零轴交叉（从负变正 = 买入）
2. MACD 线与信号线交叉
3. MACD 与价格的背离（价格创新高但 MACD 没有 → 趋势可能反转）

输出是原始数值，不做归一化。不同品种之间的 MACD 值不可直接比较。

**直觉**：

想象两条均线。当价格开始上涨时，快均线先涨，慢均线后涨，两条线的距离（MACD线）开始扩大。柱状图衡量的是 MACD 相对于自身近期均值的偏离——可以理解为"MACD 的短期动量"：

- 柱状图从负变正 = MACD 正在加速上升，趋势在增强
- 柱状图虽然为正但在缩小 = MACD 仍在上方但增速放缓，趋势在减弱

注意：柱状图不是严格数学意义上的"导数"，而是 MACD 与其自身平滑值的差。但它确实能比 MACD 线更早反映趋势的转折。

**为什么后面还要标准化？**

原始的 histogram 有个问题：它的绝对值和价格尺度有关，不同品种之间不可比，同一品种在不同波动阶段数值大小也不可直接比较。

**本项目实现**：在经典 MACD 柱状图的基础上做了归一化处理：
```
normalized = histogram / rolling_std(histogram, 63天)
signal = clip(normalized, -2, 2) / 2    → 输出范围 [-1, +1]
```
这一步不是经典 MACD 的一部分，是为了让 MACD 信号与其他信号统一到 [-1, +1] 范围、并且可以跨品种比较而做的设计选择。归一化的含义：±1 表示柱状图偏离了约 2 个标准差。

## 二、均线类

### 4. MA Crossover（SMA 交叉）

**一句话**：快均线在慢均线上方就做多。

**计算**：
```
快均线 = SMA(10天)
慢均线 = SMA(50天)
信号 = sign(快均线 - 慢均线)
```

**直觉**：均线 = 过去 N 天价格的平均值。短期均线代表"近期趋势"，长期均线代表"长期趋势"。当短期趋势超过长期趋势（快线上穿慢线），说明价格在加速上涨。

**SMA 的问题**：Simple Moving Average 对所有历史数据赋予相同权重。10天前的价格和今天的价格同等重要，这不太合理。

**本项目实现**：与经典定义一致，输出离散值 +1/-1/0。

---

### 5. EMA Crossover（EMA 交叉）

**一句话**：和 SMA 交叉一样，但用指数均线，对近期价格更敏感。

**EMA vs SMA 的区别**：

```
SMA(10) = (P1 + P2 + ... + P10) / 10     ← 每天权重相同
EMA(10) = 近期价格权重高，远期权重指数衰减  ← 今天最重要，10天前几乎不算
```

EMA 的权重衰减公式：`α = 2/(N+1)`，每天的权重是前一天的 `(1-α)` 倍。

**优点**：对新信息反应更快，信号更及时。
**缺点**：也更容易被短期噪音误导。

**经典用法**：与 SMA 交叉一样，输出二值信号——快 EMA 在慢 EMA 上方 → 做多，反之做空。

**本项目实现**：输出**连续值**而非二值信号，这与经典定义不同。具体做法：
```
diff = EMA_fast - EMA_slow
normalized = diff / rolling_std(diff, 63天)
signal = clip(normalized, -2, 2) / 2    → 输出范围 [-1, +1]
```
差距越大 → 信号绝对值越大 → 仓位越大。这样做的好处是能区分"刚刚交叉"（弱信号）和"已经分离很远"（强信号）。但这是项目的设计选择，不是 EMA 交叉的经典定义。

## 三、通道突破类

### 6. Donchian Breakout（Donchian 通道突破）

**经典定义**：

Donchian 通道由 Richard Donchian 提出，后被 Richard Dennis 的"海龟交易法则"采用。

```
上轨 = 过去 N 天最高价
下轨 = 过去 N 天最低价
```

**经典用法是二值突破信号**：
- 价格突破上轨（创 N 日新高）→ 入场做多
- 价格突破下轨（创 N 日新低）→ 入场做空
- 两次突破之间 → 维持原有持仓

海龟系统还有更多规则：用 20 日突破入场、10 日突破反向出场、基于 ATR 的仓位管理等。

**直觉**：如果价格突破了过去 20 天的最高点，说明市场正在做一些"不寻常"的事情，可能是新趋势的开始。

**本项目实现**：与经典 Donchian 有本质区别。我们没有做二值突破，而是算价格在通道内的相对位置：
```
position = (close - lower) / (upper - lower)    → 0到1之间
signal = position × 2 - 1                       → 映射到 [-1, +1]
```
含义：+1 = 通道顶部，0 = 通道中间，-1 = 通道底部。这是连续的"通道位置"指标，不是经典的突破信号。经典 Donchian 只在突破那一刻产生信号，我们的版本在通道内任意位置都有信号强度。

**和动量的区别**：动量只看起点和终点，Donchian 看的是极值。如果价格从 100 涨到 110 再跌到 105，动量说"做多"（105 > 100），但我们的 Donchian 实现会说"中性偏多"（105 在 100-110 的中间偏上）。

---

### 7. Bollinger Breakout（布林带）

**经典定义**：

由 John Bollinger 提出。

```
中轨 = SMA(20天)
上轨 = 中轨 + 2 × 标准差(20天)
下轨 = 中轨 - 2 × 标准差(20天)
```

布林带会根据市场波动率自动调整宽度。波动小的时候带变窄，波动大的时候带变宽。

**经典用法主要是均值回归**：
- 价格触及上轨 → 超买，考虑卖出或减仓
- 价格触及下轨 → 超卖，考虑买入
- Bollinger 本人强调这不是一个单独的交易系统，需要配合其他指标确认

也有人把布林带用于趋势判断（价格持续沿上轨运行 = 强趋势），但这不是主流用法。

**本项目实现**：我们把布林带用作**趋势跟踪信号**，这与经典的均值回归用法方向相反：
```
position = (close - lower) / (upper - lower)    → 0到1之间
signal = clip(position × 2 - 1, -1, +1)
```
价格在上轨附近 → 信号接近 +1 → 做多。经典用法在同样的位置会考虑做空。

这样做在趋势跟踪框架里是合理的（强趋势中价格会持续沿布林带上轨运行），但你需要知道这和教科书里的布林带用法是相反的。

**和 Donchian 的区别**：Donchian 用最高/最低价定义通道（形状由极端价格决定），布林带用标准差（形状由价格的离散程度决定）。在低波动期，布林带的通道比 Donchian 窄，更容易触发信号。

---

### 8. Keltner Breakout（Keltner 通道）

**经典定义**：

```
中轨 = EMA(20天)
ATR  = 过去20天的平均真实波幅
上轨 = 中轨 + 2 × ATR
下轨 = 中轨 - 2 × ATR
```

**ATR 是什么？** True Range = max(当日最高-最低, |最高-昨收|, |最低-昨收|)。它衡量的是每天价格实际波动的幅度，包括了日内波动和跳空缺口。

**经典用法**：与布林带类似，可用于均值回归或趋势判断，视交易者的策略而定。

**布林带 vs Keltner 的核心区别**：
- 布林带用**收盘价的标准差**定义宽度 → 只反映收盘价的分布
- Keltner 用 **ATR** 定义宽度 → 反映日内高低波动和跳空幅度
- 这意味着一天内大幅震荡但收盘回到原点的情况，布林带宽度几乎不变，但 Keltner 会变宽

**本项目实现**：与 Donchian、Bollinger 的处理方式一致——计算价格在通道中的相对位置，映射到 [-1, +1]，作为趋势跟踪信号。同样不是经典的突破或均值回归用法。

## 四、回归类

### 9. Linear Regression Slope（线性回归斜率）

**一句话**：在过去 N 天的价格上画一条"最佳拟合直线"，斜率为正就做多。

**经典定义**：
```
对过去 N 天的价格做线性回归 y = a + b×t
斜率 b > 0 → 上升趋势
斜率 b < 0 → 下降趋势
```

**R² 是什么？** 决定系数，衡量"价格变化中有多少比例可以被线性趋势解释"。

- R² = 1.0：价格完美地沿直线运动（纯趋势）
- R² = 0.5：一半趋势一半噪音
- R² = 0.0：完全随机，没有趋势

**和动量的区别**：动量只看起点终点，线性回归考虑了中间路径是否"走得直"。

**本项目实现**：在经典斜率基础上做了两层处理：

第一层 — 加入 R² 作为置信度权重：
```
raw_signal = (slope / y_std) × R²
```
斜率除以标准差是为了跨品种可比。乘以 R² 是为了在趋势线性度低时自动降低信号强度。这一步是合理的设计选择，但不是线性回归斜率信号的标准做法。

第二层 — 滚动标准差归一化：
```
normalized = raw_signal / rolling_std(raw_signal, 252天)
signal = clip(normalized, -2, 2) / 2    → [-1, +1]
```
与 MACD、EMA crossover 类似的归一化处理。

## 五、趋势强度类

### 10. ADX/DMI（Average Directional Index）

**一句话**：先判断趋势方向（+DI vs -DI），再用 ADX 衡量趋势有多强。

**经典定义（分三步）**：

**Step 1 — 方向运动 (Directional Movement)**：
```
+DM = 今日最高 - 昨日最高（如果 > 0 且 > 下跌幅度）
-DM = 昨日最低 - 今日最低（如果 > 0 且 > 上涨幅度）
```
核心思想：哪个方向的突破更大，就算哪个方向的运动。

**Step 2 — 方向指标 (Directional Indicator)**：
```
+DI = 100 × smoothed(+DM) / smoothed(ATR)    ← 上涨方向运动占总波动的比例
-DI = 100 × smoothed(-DM) / smoothed(ATR)    ← 下跌方向运动占总波动的比例
```
+DI > -DI → 多头主导，反之空头主导。两个值的范围都是 0-100。

**Step 3 — ADX（趋势强度）**：
```
DX = 100 × |+DI - -DI| / (+DI + -DI)   ← 多空力量差距有多大
ADX = smoothed(DX)                       ← 消除噪音
```
ADX 范围也是 0-100。

**经典用法**：用阈值规则判断——ADX > 25 表示有趋势（适合趋势跟踪策略），ADX < 20 表示震荡（适合均值回归策略）。方向看 +DI 和 -DI 的大小关系。ADX 本身只衡量趋势强度，不区分方向。

**独特之处**：ADX 是少数显式区分"方向"和"强度"的指标之一（linreg 的斜率+R² 也做了类似的事）。这意味着在震荡市场，即使 +DI > -DI（有微弱方向性），ADX 也会很低 → 告诉你"方向不可信，别下注"。

**本项目实现**：把经典的阈值规则改成了连续信号：
```
direction = sign(+DI - -DI)           ← +1 或 -1
adx_normalized = clip(ADX / 50, 0, 1) ← ADX 映射到 0~1
signal = direction × adx_normalized   ← 输出 [-1, +1]
```
这样 ADX=50 时信号满强度，ADX=25 时半强度，ADX 接近 0 时信号几乎为零。除以 50（而非 100）是一个经验性的选择——因为 ADX 很少超过 60，除以 100 会让大部分信号太弱。

## 信号对比总结

| 信号 | 经典输出 | 本项目输出 | 主要改动 |
|------|---------|-----------|---------|
| momentum | 离散 +1/-1 | 离散 +1/-1/0 | 无 |
| combined_momentum | — | 连续 [-1,+1] | 非经典信号，工程实践 |
| macd | 原始柱状图值 | 连续 [-1,+1] | 滚动标准差归一化 |
| ma_crossover | 离散 +1/-1 | 离散 +1/-1/0 | 无 |
| ema_crossover | 离散 +1/-1 | 连续 [-1,+1] | 归一化为连续值 |
| donchian | 二值突破信号 | 连续 [-1,+1] | 改为通道位置指标 |
| bollinger | 均值回归信号 | 连续 [-1,+1] | 改为趋势方向，且连续化 |
| keltner | 通道突破/回归 | 连续 [-1,+1] | 改为通道位置指标 |
| linreg | 斜率值 | 连续 [-1,+1] | 加 R² 权重 + 归一化 |
| adx | 阈值规则(>25) | 连续 [-1,+1] | 方向×强度合成连续信号 |

## 为什么做这些改动？

本项目把所有信号统一到 [-1, +1] 连续值，主要出于三个原因：

1. **可比性**：不同信号的输出范围统一后，才能做等权平均、加权组合等操作
2. **仓位连续性**：连续值 × ATR sizing 产生平滑的仓位变化，减少不必要的换手
3. **信号强度**：连续值能区分"强趋势"和"弱趋势"，而不是全仓或空仓

代价是偏离了经典定义。如果你在其他地方看到这些指标的介绍，它们的用法和行为可能与本项目的实现不同。

## 为什么它们高度相关？

从相关性矩阵看到这 10 个信号的相关性在 0.5-0.9 之间，这不意外：

**它们都在度量同一个东西** — 价格趋势。只是度量方式不同（就像温度计可以用水银、红外、热电偶，但测的都是温度）。在强趋势期间，所有信号都会给出相同的方向；区别主要在趋势的开始和结束阶段，不同信号的反应时间不同。

真正的分散化需要来自**不同类型的因子**，而不是同一类因子的不同实现。
