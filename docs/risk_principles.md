# 风控与组合优化原理详解

## 核心认知

> "风控比信号更重要"

这句话的意思是：**一个平庸的信号加上好的风控，表现通常优于一个优秀的信号没有风控。** 因为风控决定了你在错误的时候亏多少，而信号只决定了你在正确的时候赚多少。

从数学上看，复合收益具有非对称性：
- 亏 50% 后需要涨 100% 才能回本（在更小的基数上恢复）
- 亏 20% 后只需要涨 25% 就能回本

这是复利的凸性效应（Jensen's inequality）——同样幅度的涨跌交替会导致净亏损：`$100 × 1.5 × 0.5 = $75`。所以控制最大回撤的价值远超提升收益率。

## 一、止损

### 追踪止损 (Trailing Stop)

**经典定义**：

追踪止损只定义退出规则：
```
多头：记录持仓以来的最高价，如果价格从最高点回落超过阈值 → 平仓
空头：记录持仓以来的最低价，如果价格从最低点反弹超过阈值 → 平仓
```

阈值可以是固定金额、固定百分比、或 ATR 倍数。至于平仓后如何重新入场，不是追踪止损本身的定义——它只管"什么时候该跑"。

经典实现通常用**最高价 (high)** 来追踪极值、用**最低价 (low)** 来判断是否触发（多头情况下），这样能更及时地捕捉到日内的回撤。

**为什么用 ATR 而不是固定比例？**

ATR 是"平均每天价格波动多少点"。不同品种的波动幅度差异巨大：
- ES（标普500期货）每天波动 ~50 点
- ZN（国债期货）每天波动 ~0.3 点

用固定比例（如"跌5%就止损"）会导致：在高波动品种上太紧（天天被止损），在低波动品种上太松（等到止损时已经亏太多）。

用 ATR 倍数，3×ATR ≈ "正常波动的3倍"，这样每个品种的止损宽度都是相对于自身波动而言的。

**ATR 倍数的选择**：
- 2×ATR：很紧，会频繁止损，降低胜率但控制单次亏损
- 3×ATR：中等，经典选择
- 5×ATR：很宽松，几乎只在极端行情才触发

**止损的代价**：在震荡市中，可能刚止损价格就反弹了——"被洗出去"。这就是为什么止损不一定提升夏普，但它显著降低了最大回撤。

**本项目实现**：与经典定义有两处不同：

1. **重入规则**：平仓后等原始信号方向翻转才重新入场。这是我们的设计选择——有的系统会在止损线不再被触及时就重入，有的用独立的入场信号。
2. **用收盘价追踪**：`tracking_high = max(tracking_high, close)`，而非用日内最高价。这是简化处理——用收盘价会让止损反应慢一拍，日内已经大幅回撤但收盘拉回来的情况不会触发止损。

## 二、波动率择时

### 波动率缩放 (Vol Scaling)

**经典定义**：

核心思想：**维持恒定的风险暴露。** 如果市场波动翻倍了，同样的仓位产生的风险也翻倍了——应该把仓位减半来保持风险预算不变。

```
调整后仓位 = 原始仓位 × (目标波动率 / 实际波动率)

例如：
  目标波动率 = 15%（年化）
  当前实际波动率 = 30%
  缩放系数 = 15% / 30% = 0.5 → 仓位减半

  当前实际波动率 = 10%
  缩放系数 = 15% / 10% = 1.5 → 仓位放大到 1.5 倍
```

理论依据很直接：你的仓位应该产生"恒定的风险"，而不是"恒定的名义金额"。这不需要对市场行为做任何假设。

**经验上的额外观察**：有研究（如 Moreira & Muir, 2017）发现高波动率时期风险溢价不会等比例增加，所以 vol scaling 除了控制风险，还可能提升风险调整后收益。但这是经验发现，不是 vol scaling 的理论基础。

**上限 (vol_cap)**：防止在异常低波动时仓位过大。如果实际波动率只有目标的 1/5，理论上应该 5 倍仓位，但这太危险——低波动可能随时暴增（如 2020 年 3 月）。

**与 ATR sizing 的重叠**：注意我们已经在用 ATR-based position sizing，它本身就根据波动率调整手数。Vol scaling 在此基础上又做了一次波动率调整，存在"双重调整"问题。逻辑上不矛盾（ATR sizing 标准化的是品种间的风险，vol scaling 标准化的是时间维度的风险），但效果可能有重叠。

**本项目实现**：

使用前一天的已实现波动率计算缩放系数（`shift(1)`），避免前视偏差——实际交易中你不可能知道当天收盘后的波动率。

### 组合波动率目标 (Vol Targeting)

**与 Vol Scaling 的区别**：
- Vol Scaling：在每个品种独立调整
- Vol Targeting：先算组合整体的波动率，再统一缩放所有仓位

**为什么要在组合层面做？** 因为品种之间有相关性。如果 4 个品种都在同时加仓（因为各自波动率低），组合层面的风险可能已经超标了。组合层面的 Vol Targeting 能捕捉到这种"集体风险积累"。

**本项目实现**：与 Vol Scaling 一样使用 `shift(1)` 避免前视偏差。

## 三、组合权重分配

### 等权 (Equal Weight)

```
每个品种的风险预算 = 总风险 / N
```

**优点**：简单、不需要估计任何参数。
**缺点**：忽略了品种间的差异。高波动品种（如 CL）和低波动品种（如 ZN）分到同样的风险预算，但实际风险贡献不同。

不过，我们已经在用 ATR-based position sizing，它会根据波动率调整手数。所以等权 + ATR sizing 已经在一定程度上实现了风险标准化。

### 逆波动率权重 (Inverse Vol)

**一句话**：波动率低的品种分更多风险预算。

```
权重_i = (1 / σ_i) / Σ(1 / σ_j)

例如 4 个品种波动率分别是 20%, 30%, 15%, 10%:
  逆波动率: 5, 3.33, 6.67, 10
  归一化权重: 20%, 13%, 27%, 40%
```

**和风险平价的关系**：逆波动率权重是风险平价在"所有品种两两不相关"时的特例。当相关性为零时，协方差矩阵是对角矩阵，风险平价退化为逆波动率。

**和 ATR sizing 的重叠**：ATR sizing 已经做了类似的事——波动大的品种自动持仓少。所以逆波动率权重 + ATR sizing 有一定的"双重调整"问题，这解释了为什么它在我们的回测中没有明显改善。

### 风险平价 (Risk Parity)

**经典定义**：

**一句话**：让每个品种对组合总风险的**风险贡献 (Risk Contribution)** 相等。

注意区分两个概念：
```
边际风险贡献 MRC_i = ∂σ_p / ∂w_i = (Σw)_i / σ_p
    → 权重增加 1 单位带来的风险增量

风险贡献 RC_i = w_i × MRC_i = w_i × (Σw)_i / σ_p
    → 该品种在当前权重下实际贡献的风险量
```

风险平价的目标是等化 **RC**（风险贡献），不是 MRC（边际风险贡献）。两者的区别：MRC 相等意味着"再多买一点任何品种带来的风险增量一样"，RC 相等意味着"每个品种当前贡献的风险总量一样"。

**数学本质**：

组合波动率 σ_p = √(w'Σw)，其中 Σ 是协方差矩阵。

RC_1 + RC_2 + ... + RC_n = σ_p（风险贡献可加且恰好等于组合波动率）

风险平价的目标：RC_i = σ_p / N，对所有 i 成立。

**为什么相关性重要？**

假设 ES 和 CL 高度正相关（同涨同跌）：
- 逆波动率会给它们各自独立分配权重
- 实际上它们在组合中的风险是叠加的（相关性高 → 不分散）
- 风险平价会识别到这一点，降低这两个的总权重

假设 GC 和 ZN 与其他品种相关性低：
- 它们提供真正的分散化
- 风险平价会给它们更高的权重

**本项目实现**：

求解方法：不动点迭代 `w_i ∝ 1/(Σw)_i`，从等权起步，通常 10-20 次收敛。参考 Maillard, Roncalli & Teïlétché (2010)。

协方差矩阵用 120 天滚动窗口的样本协方差估计，年化处理（×252）。权重用 `shift(1)` 避免前视偏差。

**在我们的回测中为什么风险平价表现差？** 几个原因：
1. 只有 4 个品种，样本协方差矩阵估计不稳定
2. 品种间相关性剧烈变化（从 -0.75 到 +0.75），导致权重大幅波动
3. ATR sizing 已经做了波动率标准化，再加风险平价可能过度调整

## 四、实验结果解读

从我们的回测看：

| 方案 | 夏普 | 最大回撤 | 特点 |
|------|------|---------|------|
| 基线（无风控） | 0.91 | -9.47% | 参考线 |
| 追踪止损 | 0.67 | **-6.02%** | 夏普降了但回撤最小 |
| 波动率择时 | 0.84 | -9.72% | 变化不大 |
| 组合波动率目标 | **0.93** | -11.74% | 夏普最高但回撤更大 |
| 逆波动率 | 0.73 | -9.80% | 和基线差不多 |
| 风险平价 | 0.25 | -17.26% | 反而更差 |
| 全套 | 0.59 | -14.31% | 叠加太多不一定好 |

**关键教训**：

1. **止损的价值在回撤，不在夏普。** 夏普从 0.91 降到 0.67，但最大回撤从 -9.47% 降到 -6.02%。如果你是真金白银交易，能承受的回撤是有限的，止损是必需品。

2. **组合波动率目标最有效。** 它把夏普从 0.91 提升到 0.93，原理是在低波动时放大仓位，维持恒定的风险暴露。

3. **不要盲目叠加风控。** "全套风控"表现反而不如单项。每种风控机制都有自己的逻辑假设，叠加在一起可能相互矛盾（比如止损在减仓，vol target 在加仓）。

4. **风险平价在少品种时不靠谱。** 4 个品种的协方差矩阵太不稳定，等扩展到 10+ 品种后（项目四）再试。

5. **ATR sizing 本身就是一种风控。** 它已经在做波动率标准化，所以额外的逆波动率权重和 vol scaling 效果有限——存在双重调整。

## 五、下一步思考

看完这些结果，可以思考：

1. **止损 vs 收益的权衡**：你愿意用多少夏普来换更小的回撤？这取决于你的风险偏好和资金规模。

2. **波动率目标的风险**：Vol Target 在低波动时放大仓位，但"低波动"可能突然结束（2020年3月）。vol_cap 参数就是防这个的。

3. **参数选择**：止损 3×ATR、波动率目标 10%——这些参数本身也可以优化，但要小心过拟合。

4. **品种数量的重要性**：很多风控技术（尤其是风险平价）在品种多的时候才能发挥作用。这指向项目四（跨市场扩展）的必要性。
