# 风控与组合优化原理详解

## 核心认知

> "风控比信号更重要"

这句话的意思是：**一个平庸的信号加上好的风控，表现通常优于一个优秀的信号没有风控。** 因为风控决定了你在错误的时候亏多少，而信号只决定了你在正确的时候赚多少。

从数学上看，收益是加法的，但亏损是乘法的：
- 亏 50% 后需要涨 100% 才能回本
- 亏 20% 后只需要涨 25% 就能回本

所以控制最大回撤的价值远超提升收益率。

## 一、止损

### 追踪止损 (Trailing Stop)

**一句话**：持仓期间价格从最高点回撤超过阈值就平仓。

**机制**：
```
多头持仓中：
  记录持仓期内最高价 (tracking_high)
  如果 tracking_high - 当前价格 > 3 × ATR → 平仓
  平仓后等信号方向翻转才重新入场

空头持仓中：
  记录持仓期内最低价 (tracking_low)
  如果 当前价格 - tracking_low > 3 × ATR → 平仓
```

**为什么用 ATR 而不是固定比例？**

ATR 是"平均每天价格波动多少点"。不同品种的波动幅度差异巨大：
- ES（标普500期货）每天波动 ~50 点
- ZN（国债期货）每天波动 ~0.3 点

用固定比例（如"跌5%就止损"）会导致：在高波动品种上太紧（天天被止损），在低波动品种上太松（等到止损时已经亏太多）。

用 ATR 倍数，3×ATR ≈ "正常波动的3倍"，这样每个品种的止损宽度都是相对于自身波动而言的。

**ATR 倍数的选择**：
- 2×ATR：很紧，会频繁止损，降低胜率但控制单次亏损
- 3×ATR：中等，经典选择
- 5×ATR：很宽松，几乎只在极端行情才触发

**止损的代价**：每次止损后必须等信号翻转才能重新入场。在震荡市中，可能刚止损价格就反弹了——"被洗出去"。这就是为什么止损不一定提升夏普，但它显著降低了最大回撤（从 -9.47% 降到 -6.02%）。

## 二、波动率择时

### 单品种波动率择时 (Vol Scaling)

**一句话**：市场波动大时减仓，波动小时加仓。

**公式**：
```
调整后仓位 = 原始仓位 × (目标波动率 / 实际波动率)

例如：
  目标波动率 = 15%（年化）
  当前实际波动率 = 30%
  缩放系数 = 15% / 30% = 0.5 → 仓位减半

  当前实际波动率 = 10%
  缩放系数 = 15% / 10% = 1.5 → 仓位放大到 1.5 倍
```

**直觉**：你的仓位应该产生"恒定的风险"，而不是"恒定的名义金额"。如果市场波动翻倍了，同样的仓位产生的风险也翻倍了——你应该把仓位减半来保持风险不变。

**为什么有效？** 高波动率往往伴随着趋势反转或震荡，此时趋势信号失效概率高。低波动率时趋势信号更可靠，可以放大仓位。

**上限 (vol_cap = 2.0)**：防止在异常低波动时仓位过大。如果实际波动率只有目标的 1/5，理论上应该 5 倍仓位，但这太危险——低波动可能随时暴增。

### 组合波动率目标 (Vol Targeting)

**与单品种 Vol Scaling 的区别**：
- Vol Scaling：在每个品种独立调整
- Vol Targeting：先算组合整体的波动率，再统一缩放所有仓位

**为什么要在组合层面做？** 因为品种之间有相关性。如果 4 个品种都在同时加仓（因为各自波动率低），组合层面的风险可能已经超标了。组合层面的 Vol Targeting 能捕捉到这种"集体风险积累"。

## 三、组合权重分配

### 等权 (Equal Weight)

```
每个品种的风险预算 = 总风险 / N
```

**优点**：简单、不需要估计任何参数。
**缺点**：忽略了品种间的差异。高波动品种（如 CL）和低波动品种（如 ZN）分到同样的风险预算，但实际风险贡献不同。

不过，我们已经在用 ATR-based position sizing，它会根据波动率调整手数。所以等权 + ATR sizing 已经在一定程度上实现了风险标准化。

### 逆波动率权重 (Inverse Vol)

**一句话**：波动率低的品种分更多风险预算。

```
权重_i = (1 / σ_i) / Σ(1 / σ_j)

例如 4 个品种波动率分别是 20%, 30%, 15%, 10%:
  逆波动率: 5, 3.33, 6.67, 10
  归一化权重: 20%, 13%, 27%, 40%
```

**和 ATR sizing 的关系**：ATR sizing 已经做了类似的事——波动大的品种自动持仓少。所以逆波动率权重 + ATR sizing 有一定的"双重调整"问题，这解释了为什么它在我们的回测中没有明显改善。

### 风险平价 (Risk Parity)

**一句话**：让每个品种对组合总风险的"边际贡献"相等。

**和逆波动率的区别**：逆波动率只看各品种自身的波动率，不考虑品种之间的相关性。风险平价考虑了相关性。

**为什么相关性重要？**

假设 ES 和 CL 高度正相关（同涨同跌）：
- 逆波动率会给它们各自独立分配权重
- 实际上它们在组合中的风险是叠加的（相关性高 → 不分散）
- 风险平价会识别到这一点，降低这两个的总权重

假设 GC 和 ZN 与其他品种相关性低：
- 它们提供真正的分散化
- 风险平价会给它们更高的权重

**数学本质**：

组合波动率 σ_p = √(w'Σw)，其中 Σ 是协方差矩阵。

品种 i 的风险贡献 = w_i × (Σw)_i / σ_p

风险平价的目标：所有品种的风险贡献相等 = σ_p / N

**在我们的回测中为什么风险平价表现差？** 几个原因：
1. 只有 4 个品种，协方差矩阵估计不稳定
2. 品种间相关性剧烈变化（从 -0.75 到 +0.75），导致权重大幅波动
3. ATR sizing 已经做了波动率标准化，再加风险平价可能过度调整

## 四、实验结果解读

从我们的回测看：

| 方案 | 夏普 | 最大回撤 | 特点 |
|------|------|---------|------|
| 基线（无风控） | 0.91 | -9.47% | 参考线 |
| 追踪止损 | 0.67 | **-6.02%** | 夏普降了但回撤最小 |
| 波动率择时 | 0.84 | -9.72% | 变化不大 |
| 组合波动率目标 | **0.93** | -11.74% | 夏普最高但回撤更大 |
| 逆波动率 | 0.73 | -9.80% | 和基线差不多 |
| 风险平价 | 0.25 | -17.26% | 反而更差 |
| 全套 | 0.59 | -14.31% | 叠加太多不一定好 |

**关键教训**：

1. **止损的价值在回撤，不在夏普。** 夏普从 0.91 降到 0.67，但最大回撤从 -9.47% 降到 -6.02%。如果你是真金白银交易，能承受的回撤是有限的，止损是必需品。

2. **组合波动率目标最有效。** 它把夏普从 0.91 提升到 0.93，原理是在低波动时放大仓位（而低波动期恰好是趋势信号最有效的时候）。

3. **不要盲目叠加风控。** "全套风控"表现反而不如单项。每种风控机制都有自己的逻辑假设，叠加在一起可能相互矛盾。

4. **风险平价在少品种时不靠谱。** 4 个品种的协方差矩阵太不稳定，等扩展到 10+ 品种后（项目四）再试。

5. **ATR sizing 本身就是一种风控。** 它已经在做波动率标准化，所以额外的逆波动率权重效果有限。

## 五、下一步思考

看完这些结果，可以思考：

1. **止损 vs 收益的权衡**：你愿意用多少夏普来换更小的回撤？这取决于你的风险偏好和资金规模。

2. **波动率目标的风险**：Vol Target 在低波动时放大仓位，但"低波动"可能突然结束（2020年3月）。vol_cap 参数就是防这个的。

3. **参数选择**：止损 3×ATR、波动率目标 10%——这些参数本身也可以优化，但要小心过拟合。

4. **品种数量的重要性**：很多风控技术（尤其是风险平价）在品种多的时候才能发挥作用。这指向项目四（跨市场扩展）的必要性。
