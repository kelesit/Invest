# Carry 因子原理

## 一、什么是 Carry

Carry 是持有某个头寸的**隐含收益**——不考虑价格涨跌，光是"拿着"就会赚或亏的部分。

在期货市场，carry 体现为**期限结构**（term structure）：近月合约和远月合约的价差。

### Backwardation（现货溢价）

近月 > 远月。持有多头有利——你买入近月合约，到期前换到更便宜的远月合约，赚取价差。

经济学解释：现货需求旺盛（如原油供应短缺），持有实物商品有"便利收益"（convenience yield），使得近月价格高于远月。

### Contango（期货溢价）

远月 > 近月。持有多头不利——每次换月都要付出价差（roll cost）。

经济学解释：远月价格反映了存储成本（仓储、保险、利息）。在供给充裕时，持有实物商品的便利收益低于存储成本，导致远月贵于近月。

### 不同品种的 carry 特征

| 品种 | 典型状态 | 经济学原因 |
|------|---------|-----------|
| 原油 (CL) | 常 contango，偶 backwardation | 存储成本高；供应中断时切换到 backwardation |
| 黄金 (GC) | 几乎总 contango | 持有成本 ≈ 无风险利率（黄金无便利收益） |
| 国债 (ZN) | 取决于收益率曲线斜率 | 正常时近月>远月（正收益率曲线），倒挂时反转 |
| 股指 (ES) | 通常微弱 contango | 远月反映融资成本减去股息率 |

## 二、Carry 信号的经典定义

### 截面 Carry（学术经典）

**经典定义**（Koijen et al., 2018 "Carry"）：

```
carry_i = (F_near_i - F_far_i) / F_near_i
```

截面策略：在同一时间点，对所有品种计算 carry 值，做多 carry 最高的品种、做空 carry 最低的品种。这是一个**相对排序**策略——不关心单个品种的 carry 绝对值，只关心它在所有品种中排第几。

截面 carry 的优势：
- 品种越多，信号越稳定（排序比绝对值更鲁棒）
- 天然市场中性（多空对冲）
- 学术文献中通常用 20-50 个品种

### 时间序列 Carry

对单个品种做多/做空决策：carry 为正（backwardation）则做多，carry 为负（contango）则做空。

这比截面版本更激进——你在赌单个品种的期限结构方向。

## 三、本项目实现

### 实现方式

```python
raw_carry = (front_close - back_close) / front_close
```

用 c.0（主力合约）和 c.1（次主力合约）的原始收盘价（未做 ratio/panama 调整）。

归一化：用 252 天滚动标准差归一化到 [-1, +1]，使信号与趋势信号接口一致。

### 与经典定义的差异

| | 经典定义 | 本项目实现 |
|--|---------|-----------|
| **策略类型** | 截面（跨品种排序） | 时间序列（单品种方向） |
| **品种数量** | 20-50 | 4 |
| **合约选择** | 通常用特定到期月 | c.0 和 c.1 连续合约 |
| **归一化** | 截面 z-score | 时间序列滚动标准差 |

### 实验结论

**Carry 信号与趋势信号低相关（avg |r| = 0.03-0.16 vs 趋势间 0.63）**。这验证了"不同数据源产生低相关信号"的假设。

**但 carry 在 4 个品种上单独亏钱（Sharpe -1.30）**，组合后拖累趋势表现。

原因分析：
1. **品种太少**：4 个品种无法构建有效的截面策略
2. **时间序列 carry 噪音大**：单品种的期限结构信号不够稳定
3. **样本期特殊性**：2019-2026 年包含 COVID 冲击和极端通胀，期限结构剧烈波动

## 四、核心教训

**低相关 ≠ 有价值。** 多因子组合要有效，需要两个条件同时满足：

1. 因子之间低相关（分散化条件）✓
2. 每个因子本身有正的期望收益（alpha 条件）✗

只满足条件 1 不满足条件 2，等于往好策略里掺噪音。
