<--
**业界从来不是在用单个 OLS 回归组合因子，而是在用更复杂的非线性模型把多个弱信号组合起来**。
核心发现有三个：

- 第一，非线性模型显著超过线性

- 第二，所有模型一致认可的主导特征几乎全是量价类——动量（1 个月、12 个月、换手率加权动量）、流动性（bid-ask spread 代理、Amihud 的 ILLIQ、异常成交量）、波动率（总波动率、特异波动率、最大日收益率）。这个事实本身是对整个量价因子研究传统的巨大背书：**用 ML 自动挑选最有用的特征，挑出来的还是这套东西**。

- 第三，神经网络的增量主要来自捕捉非线性的特征交互，不是引入新信息。换句话说，ML 带来的不是"新的信号源"，而是"更好的组合方式"。


**来源：Shihao Gu、Bryan Kelly、Dacheng Xiu 2020 年在《Review of Financial Studies》发表的"Empirical Asset Pricing via Machine Learning"**

**近期前沿研究**
- formulaic alpha 自动挖掘
    1. AlphaGen（arXiv:2306.12964）是里程碑工作，第一次把 alpha 生成建模为马尔可夫决策过程——用逆波兰表达式（RPN）作为 alpha 的内部表示，用 PPO 算法在算子空间上搜索，奖励信号不是单因子 IC 而是"新 alpha 加入现有组合后下游模型的提升"。这个奖励函数设计极其重要——它避免了搜索到一堆高 IC 但互相冗余的因子。代码开源于 RL-MLDM/alphagen GitHub 仓库。
    2. RiskMiner（arXiv:2402.07080）用蒙特卡洛树搜索替代 PPO
    3. Alpha²（arXiv:2406.16505）强调逻辑可解释的 alpha 生成
    4. AlphaForge（arXiv:2406.18394）提出"生成-预测两阶段网络 + 动态因子加权"
    5. QuantFactor REINFORCE（arXiv:2409.05144）用方差受限 REINFORCE 解决 alpha 信号不稳定问题
    6. AlphaAgent（arXiv:2502.16789）让 LLM 作为 alpha 生成器并加入针对 alpha decay 的正则化
    7. AlphaEval（arXiv:2508.13174）给这个子领域提供了统一的评测协议。一个值得警惕的事实是：AlphaEval 在对比 GP、AlphaGen、AlphaQCM 等方法时发现，**不同方法之间的差异比各自论文宣称的要小，且对超参数高度敏感**

- Transformer 与序列模型
    1. MASTER（arXiv:2312.15235，AAAI 2024）是目前 A 股截面 transformer 的事实基线，结构是市场状态门控 + 双向 attention（intra-stock 时间维 + inter-stock 截面维）。
    2. 微软亚研 2025 年的 RD-Agent-Quant（arXiv:2505.15155）更进一步，用 LLM 多 agent 自动迭代"提假设 → 写代码 → 回测 → 改进"的全研发链路，声称在 A 股上比经典因子库翻倍且只用 70% 的特征数。
    3. Bryan Kelly 团队的工作——Kelly, Kuznetsov, Malamud, Xu 2024 年的"Artificial Intelligence Asset Pricing Models"（NBER w33351），把 transformer 嵌入 SDF，声称跨资产 attention 能显著降低定价误差并提升样本外 Sharpe。这条线和 MASTER 系几乎不互相引用——一条是工程派，一条是理论派，两条并行。
- Graph Neural Network**。用行业关系、供应链关系、持股关系、甚至新闻共现关系构造股票之间的图，再用 GCN/GAT 做信息传递。

**第一，经典量价因子作为风险溢价或行为偏差的来源仍然存在**，但样本外溢价显著低于发表期水平。动量、低波、流动性、短期反转都还活着，但年化 alpha 从 1990 年代的 5-8% 下降到今天的 1-3%。

**第二，单一因子的 alpha 几乎已被套利完，alpha 来自因子组合方式与执行细节**。McLean-Pontiff 2016 的 58% 发表后衰减是这句话的量化证据。组合方式的重要性超过任何单一信号的重要性。

**第三，ML 提升的核心来自非线性交互与高维度信号融合**，而不是新数据源。Gu-Kelly-Xiu 2020 反复强调了这一点——深度学习的增量来源不是"发现了人类想不到的信号"，而是"把已知的弱信号以非线性方式组合得更好"。

**第四，严格的 multiple testing、purged CV、anti-snooping 纪律是研究的最低标准**。Lopez de Prado 的《Advances in Financial Machine Learning》2018 年出版后，purged K-fold、CPCV（Combinatorial Purged Cross-Validation）成为任何严肃 ML 研究必须使用的方法。2024 年 Arian、Norouzi、Seco 在合成可控环境中对比 walk-forward、K-fold、Purged K-fold 和 CPCV（SSRN 4686376），结论是 **CPCV 在 PBO 和 Deflated Sharpe 上显著优于其他方法**。

**第五，真正能持续产生 alpha 的不是低频学术因子，而是 Renaissance、Two Sigma、WorldQuant、幻方、九坤这类短持有期 + 容量受限 + 执行驱动的中高频量价策略**。低频学术因子产品（DFA、AQR、smart beta ETF）提供的是风险溢价和 β+，不是真正意义上的 alpha。

**价格序列的确包含截面可预测信息，但这些信息的 alpha 幅度很小，容易被套利吃掉，唯一可持续的方法是在严格防过拟合纪律下做大量弱信号的非线性组合，并把执行成本做到极致**。

<--
**量价因子流派的生态位**
5-60 日横截面、全市场宽广 universe、低成本快速迭代
最稳健留存的反而是 size、momentum、illiquidity、idiosyncratic vol 这些纯量价构造

中国市场的特殊性
**过去 20 年 A 股量价类与小市值因子的表现远强于基本面因子**（财报质量低、再融资套利普遍、散户主导导致价量信号衰减更慢）。截至 2025 年 11 月，全市场 833 只股票量化多头产品平均超额 +17.25%，91.48% 产品有正超额——这种超额规模在美股已不可想象，反映的是 A 股仍处于"量价因子套利红利期"。
中国是量价因子绝对主导。**同样是用量价做股票截面，在美国是被挤压的中间地带，在中国是主战场**。
